Welcome to Leather Pattern Designer

Design saddle-stitched leather patterns and print them at true physical scale.

Quick Start

draw → stitch → print, in four steps
Step 1

Draw a piece

Pick the Rectangle tool (B) or the Pen (P) and drag on the canvas. With the Pen, click for corners and click-drag for smooth curves; click the first point again to close the shape. Everything is measured in real millimetres.

Step 2

Select & shape it

Switch to Select (V), click a piece, then drag the handles to resize or the corner ring to rotate. The Properties panel sets exact size, rounded corners and colour. Alt-drag makes a copy; Ctrl+D duplicates.

Step 3

Add saddle stitching

With a piece selected, tick “Stitch all edges” in the Stitching panel — or click a single edge first to stitch just that side. Set the margin (distance from the edge) and spacing (your pricking-iron size); holes are evenly distributed with a stitch forced at every corner.

Step 4

Print at true scale

Use File ▸ Print and pick your paper size — the pattern prints 1:1 in millimetres, one sheet per artboard, ready to cut out and trace onto leather. File ▸ Export SVG hands the vector off to a cutter.

Want more? Add text labels (T), multiple artboards (A), and organise pieces in the Layers panel.

Stitching Workflow
Select a shape → the Stitching panel appears. Toggle Stitch all edges, or click a single edge and toggle just that side. Set the margin (inset from the edge) and spacing (hole interval) in mm. Sharp corners always get a hole (highlighted teal) so the stitch wraps cleanly — spacing into a corner may be uneven by design, matching hand-stitching. Hole style (round / diamond / french) is a global choice in Settings.
Print & Export (cut at true scale)
Ctrl+P resets to the artboard and prints it at true physical size (A4 prints A4, etc.) — UI hidden, shapes in black, stitch marks in dark red; choose “Save as PDF” in the dialog. Export SVG gives a vector file at true mm scale for Inkscape/Illustrator. .lpd (Leather Pattern Document) is the editable native save.
